## Постановка
Ищем `x` из `A x = b` для матрицы:
- `A[i,i] = 2N + 1`, `A[i,j] = 1` при `i ≠ j`.
- Выбираем `x_true` (случайный), считаем `b = A x_true`, затем «забываем» `x_true` и находим `x` итерационно.

## Метод
Метод Якоби:
```
x^{k+1}_i = (b_i − Σ_{j≠i} a_{ij} x^k_j) / a_{ii}
```
Останов: `||A x − b||_2 / ||b||_2 < eps` или `k >= maxit`.

## Параллельная реализация (MPI)
- Вектор `x` делится равномерно между процессами.
- На каждой итерации всем процессам нужен `Σ_j x_j` → одна коллективная операция `MPI_Allreduce`.
- Обновление своего блока `x` — локально.
- Невязка `||A x − b||_2` считается локально и суммируется `MPI_Allreduce`.
